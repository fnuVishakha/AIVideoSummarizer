# AIVideoSummarizer

## Overview
This repository contains the complete implementation of the project along with all required source files, documentation, and assets. The project is structured in a clean and organized manner so that anyone can easily understand, run, or extend it.

## Project Structure
The repository is organized as follows:

- `src/`  
  Contains the main source code and supporting logic files.

- `docs/`  
  Contains documentation related to the project such as descriptions and explanations.

- `assets/`  
  Contains media files used in the project, including videos.

- `README.md`  
  Provides an overview and instructions for the project.

## Assets and Video Usage
The `assets` folder contains a video file used for demonstration purposes.

- The video is **already saved inside the `assets` folder**.
- If someone wants to **try the project with a different video**, they can:
  1. Add their new video file inside the `assets` folder.
  2. Update the code to reference the new video file.
- No other changes to the project structure are required.

This makes the project flexible and easy to test with different video inputs.

## How to Use
1. Clone or download the repository.
2. you will have to use the virtual venv environment. 
3. Open the project in VS Code or any preferred editor.
4. Ensure all required dependencies are installed.
5. Run the main file from the `src` folder.
6. The project will use the video available in the `assets` folder.

## Notes
- The folder structure should be kept unchanged for proper execution.
- Additional assets or files can be added without affecting the core logic.
